# 달숨 글램핑 — 배포 직전 체크리스트

운영 배포 전에 아래만 확인하면 됩니다. (코드/메타 문구·`bookingAnchor` 정리 등은 완료된 상태로 가정)

## 환경·URL

- [ ] 프로덕션 도메인에 맞게 `NEXT_PUBLIC_SITE_URL` 설정 (또는 호스팅에서 동일 목적의 사이트 URL 변수)
- [ ] 배포 후 검색·SNS 미리보기에서 `metadataBase`/OG가 **실제 도메인** 기준으로 잡히는지 확인

## 자산·연락처

- [ ] `public/images/og-default.jpg` — 1200×630 전용 이미지로 교체했는지, 또는 현재 파일이 그 비율·용도에 맞는지 확인
- [ ] 푸터·이용안내 등의 **연락처·계좌·주소**를 실제 운영 정보와 최종 대조

## 동작 확인

- [ ] 모바일 **진입 오버레이** (첫 방문/세션) 의도대로 동작하는지
- [ ] `tel:` 전화 링크 및 **예약 문의하기** 등 전화 CTA가 기기에서 정상 동작하는지
